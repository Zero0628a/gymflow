\section{Resumen Ejecutivo}

GymFlow es una aplicación móvil orientada a apoyar al usuario durante su sesión de entrenamiento en el gimnasio. Su propósito principal es permitir la consulta rápida de grupos musculares, ejercicios y variantes útiles cuando una máquina o implemento no se encuentra disponible, reduciendo así la improvisación y mejorando la continuidad de la rutina.

El proyecto se desarrolla con React Native y Expo, utilizando servicios de Firebase para autenticación y almacenamiento de información. El informe presenta el análisis del problema, el diseño de la solución, la arquitectura propuesta, la metodología SCRUM aplicada y la organización del trabajo grupal, estableciendo la base conceptual y técnica para la implementación del sistema.
